# Task Manager

Консольний застосунок для перегляду проєктів та їх завдань.

## Структура проєктів

- **TaskManager.Models** — класи для зберігання даних (ProjectEntity, TaskEntity) та enum'и
- **TaskManager.ViewModels** — класи для відображення та взаємодії (ProjectViewModel, TaskViewModel)
- **TaskManager.Services** — штучне сховище (FakeStorage) та сервіс доступу до нього (TaskRepository)
- **TaskManager.Console** — консольний застосунок

## Як користуватись

Після запуску:
- `1` — завантажити список проєктів
- `2` — вибрати проєкт і переглянути його завдання
- `d [ID]` — переглянути деталі конкретного завдання
- `b` — повернутись до головного меню
- `0` — вийти
